Panduan Penggunaan PDF Editor Pro

1

Pilih Alat yang Diinginkan di Menu Utama

PDF Editor Pro menyediakan 7 alat lengkap: Edit PDF & Tanda Tangan, Gabung PDF, Hapus Halaman, Kompres PDF, PDF ke Gambar, Gambar ke PDF, dan PDF ke Word (.doc).

2

Edit & Tanda Tangan Dokumen

Di menu Edit, Anda bisa melukis tanda tangan, mengetik nama dengan font sambung, atau menambahkan catatan teks dan stempel resmi (seperti DISETUJUI / LUNAS) ke atas halaman.

3

Simpan dengan Folder Pilihan (Save As)

Saat menekan tombol simpan atau ekspor, jendela dialog standar Windows akan Terbuka sehingga Anda bisa memilih direktori penyimpanan dan mengubah nama file sesuka hati!
